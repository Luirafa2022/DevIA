Exemplo 2: Função que verifica se um número é par
def is_par(numero):
return numero % 2 == 0

Exemplo 3: Função que retorna o maior valor entre dois números
def maior_valor(a, b):
if a > b:

else:

Exemplo 5: Função que calcula o fatorial de um número
def fatorial(n):
if n == 0:
return 1
else:
return n * fatorial(n - 1)
